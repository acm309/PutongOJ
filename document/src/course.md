# 课程系统

课程系统旨在帮助教师方便地管理班级、布置作业，本指南将介绍如何使用课程模块的各项功能。

## 创建课程

由于权限控制原因，普通教师账号默认无法直接创建课程。请联系**平台管理员**为你创建课程。

管理员在创建课程时需要填写：
- **课程名称**: 必填。
- **课程描述**: 选填。
- **公开 / 私有**:
  - **公开**: 任何人都可以查看。
  - **私有**: 只有成员可以查看。
- **邀请码**: 选填。如果设置了邀请码，学生输入正确的邀请码即可自动加入课程。

## 进入课程

登录后，在顶部导航栏选择**课程** (Courses)，你将看到所有的课程列表。点击你所管理的课程名称即可进入课程主页。

## 课程设置

进入课程后，点击**设置** (Settings) 选项卡。如果你拥有**管理课程 (Manage Course)**权限，通过此页面你可以修改：
- **课程名称**
- **课程描述**
- **公开 / 私有**
- **邀请码** (设置为空即可关闭通过邀请码加入的功能)

## 成员管理

点击**成员** (Members) 选项卡，你可以查看当前课程的所有成员。

### 角色与权限

作为课程管理者，你可以点击成员右侧的 **编辑** 按钮修改其权限。可用权限包括：
- **基础访问 (Basic)**: 能够访问课程内容（所有成员默认拥有）。
- **查看测试用例 (View Testcase)**: 允许查看题目测试用例详情。
- **查看题解 (View Solution)**: 允许查看题目解答。
- **管理题目 (Manage Problem)**: 允许在课程内创建新题目、修改题目信息。
- **管理比赛 (Manage Contest)**: 允许在课程内创建和管理考试 / 比赛。
- **管理课程 (Manage Course)**: 允许修改课程设置、管理成员权限。

### 移除成员

点击成员右侧的**删除**按钮可以将该用户移出课程。

## 题目管理

点击 **题目** (Problems) 选项卡，你可以查看课程内的题目列表。

### 创建新题目

如果你拥有**管理题目**权限，点击页面上方的**创建题目** (Create Problem) 按钮，即可进入题目创建页面。
创建后的题目将自动隶属于本课程，且默认只对课程成员可见。

### 引入现有题目

::: tip 注意
此功能仅限 **平台管理员 (Admin)** 使用。
:::

平台管理员可以将 OJ 题库中已有的题目添加到课程中，方便复用经典题目。

### 调整题目顺序

在题目列表中，你可以：
- 点击**排序** (Sort 图标) 调整题目在列表中的显示顺序（需要管理员或题目所有者权限）。
- 点击**状态**按钮切换题目状态（如隐藏、公开）。
- 点击**删除**按钮将题目从课程中移除（不会删除题目本身）。

## 考试 / 比赛管理

点击**比赛** (Contests) 选项卡，你可以查看课程内的所有作业和考试。

### 创建比赛

如果你拥有**管理比赛**权限，点击**创建比赛** (Create Contest) 按钮。
- **标题**: 比赛 / 作业名称。
- **开始 / 结束时间**: 设置合理的时间范围。
- **公开 / 私有**: 通常课程作业设为私有。

创建完成后，你可以像管理普通比赛一样管理课程比赛（添加题目、管理选手等）。课程比赛默认对课程成员开放。
